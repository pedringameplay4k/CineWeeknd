# 🎬 CineWeeknd

O **CineWeeknd** é uma aplicação web voltada para amantes de cinema, desenvolvida com o objetivo de oferecer uma experiência simples e intuitiva de navegação entre filmes, sinopses e informações do mundo cinematográfico.

Este projeto foi criado como parte dos meus estudos e prática em **desenvolvimento web**, aplicando conceitos de **HTML, CSS e JavaScript**, além do consumo de APIs públicas de filmes.

---

## 🚀 Funcionalidades

- 🔎 **Busca de filmes** por nome ou categoria  
- 🎞️ **Listagem dinâmica** com pôsteres, títulos e sinopses  
- ⭐ **Sistema de destaque** para filmes populares ou em cartaz  
- 💾 (Em desenvolvimento) Favoritar filmes e salvar preferências do usuário  
- 📱 **Design responsivo**, adaptável para desktop e dispositivos móveis  

---

## 🧠 Tecnologias Utilizadas

| Categoria | Tecnologias |
|------------|--------------|
| **Front-end** | HTML5, CSS3, JavaScript |
| **APIs** | [The Movie Database (TMDB)](https://www.themoviedb.org/) *(ou outra API, se você usou diferente)* |
| **Ferramentas** | Visual Studio Code, Git, GitHub |
